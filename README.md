# ServiceLink Senior Product Designer, AI - Independent Candidate Vision

This public static site presents Russell Dudek's independent candidate thesis for ServiceLink's Senior Product Designer, AI role.

## Thesis

Give every agent clear title to act: make intent, evidence, authority, exception conditions, recovery and the resulting decision receipt visible enough to trust and improve.
